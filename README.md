# 🔐 API Login

API REST desenvolvida com ASP.NET Core para autenticação de usuários utilizando JWT.

## 🚀 Tecnologias Utilizadas

- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server
- JWT Authentication
- FluentValidation

## 🧠 Arquitetura

Projeto estruturado utilizando:

- Application 
- Domain 
- Infraestrutura
- Comunication
- Exception
- Repository Pattern
- Unit Of Work

## 📌 Funcionalidades

- Cadastro de usuários
- Login de usuários
- Geração de token JWT
- Validação de autenticação
- Validação de Usuario.
- Criptografia de senha
- Validações com FluentValidation

 🔒 Segurança

- Senhas criptografadas
- Autenticação via JWT
- Rotas protegidas

## 📚 Objetivo

Projeto desenvolvido em  ASP.NET Core para treinar autenticação .
